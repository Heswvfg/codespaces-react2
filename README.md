# Список любимых песен

Простое приложение на React для управления коллекцией любимых песен.

## Как работает

1. **Добавить песню**  
   Введите название и исполнителя, нажмите "+"

2. **Редактировать**  
   Нажмите "Изм" на любой песне, измените данные, сохраните "✓" или отмените "✗"

3. **Удалить**  
   Нажмите "X" рядом с песней
